# Node Show
Node CLI to output system information.

    Usage: show [(-uon | V | h )]

    -u, --user     Current User Details
    -o, --os       OS Details
    -n, --node     Node Details
    -V, --version  Version
    -h, --help     Command Help

Based off tutorial: https://www.hacksparrow.com/nodejs/developing-command-line-tools.html
